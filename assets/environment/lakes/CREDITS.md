# Lakes Environment Assets

The Lakes stage includes these CC0 texture assets as optional material sources, with procedural canvas fallbacks in `roadGenerator.js`:

- Aerial Rocks 01 by Rob Tuytel, Poly Haven, CC0: https://polyhaven.com/a/aerial_rocks_01
- Forest Ground 04 by Rob Tuytel, Poly Haven, CC0: https://polyhaven.com/a/forest_ground_04. Included for future forest-floor/shoreline tests, but not currently used as the main Lakes terrain because it read too barren in gameplay.
