# RallyRushII

RallyRushII is a standalone continuation of RallyRush, built from the original RallyRush codebase as a separate repository for new development.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

For example:

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Repository

This repository is independent from RallyRush. Changes here are intended for RallyRushII development.
